# 제 20회 임베디드 경진대회 - 지능형 휴머노이드 부분

## 1. 대회 소개
본 대회는 재난 및 사고 상황에서 인간이 접근하기 어려운 지역에 2족 보행 로봇을 투입하여 임무를 수행하는 것을 목표로 한다. <br>
임베디드 환경 내에서 소프트웨어를 구축하여 로봇의 하드웨어를 제어하고, 영상 제어 및 통신 기술을 활용해 주어진 미션을 안정적으로 완수하는 능력을 평가한다.

## 2. 경진대회 팀 소개 및 역할
- 최영락(팀장) : OpenCV 컨투어 기반 라인트레이스 기능 구현 및 로봇 모션 제어 보조 <br>
- 강선규(팀원) : 컬러 트래킹 알고리즘 및 알파벳(A,B,C,D) 인식 기능 개발 <br>
- 곽수정(팀원) : 라즈베리 파이와 제어 보드 간 시리얼 통신 및 상화별 모션 분기 로직 구현 <br>
- 김선영(팀원) : RoboBasic을 이용한 로봇의 관절별 모션 데이터 생성 및 최적화 <br>
- 송우석(팀원) : 로봇 보행 모션 보조 및 컬러 트래킹 기능 구현 지원 <br>


## 3. 개발 환경 및 기술 스택

### 3.1 하드웨어

- **휴머노이드** : MF-RAPI4 (18관절)
- **메인보드** : Raspberry Pi 4 Model B (4G-64G 메모리탑재)
- **제어보드** : MR-C3024FX
- **센서** : 5M 픽셀 카메라(1080p 지원), 2축 기울기 센서, 2축 자이로 센서, 전방 적외선 거리감지 센서

### 3.2 소프트웨어 및 언어
- **OS** : Raspbian Linux(Raspberry Pi), 로보베이직(Control Board)
- **Development Tools** : Thonny, Geany (Raspberry Pi 내부 개발 환경)
- **Languages** : Python 3.x, 로보베이직 v2.8
- **Libraries*** : OpenCV, Numpy, PySerial

## 4. 주요 기능 및 코드 구조

### 4.1 소스 코드 개요
- **alpha.py** : OpenCV의 컨투어 및 HSV 색 공간 분석을 활용하여 노란색 라인을 추적하고, 재난 지역의 알파벳 및 방위 표지판을 인식하는 영상 처리 모듈이다.
- **motion.py** : 라즈베리 파이에서 처리된 데이터를 기반으로 제어 보드에 시리얼 신호(1byte 단위)를 송신하여 로봇의 동작을 실행하는 인터페이스 클래스이다.
- **robobasicMotion** : 로봇의 각 서브 모터 각도값을 직접 제어하여 보행, 계단 오르기, 물건 집기 등의 세부 동작을 정의한 하위 레벨 제어 프로그램이다.
- **main.py** : 영상 처리 결과에 따라 실시간으로 로봇의 이동 방향과 미션 수행 여부를 결정하는 시스템 통합 메인 루틴이다.

### 4.2 수행 미션
- **라인 트레이싱** : 노란색 가이드라인을 인식하여 경로를 이탈하지 않고 주행한다.
- **방위 및 방향 인식** : 동서남북(E, W, S, N) 표지판을 인식하여 팔 동작 및 음성 출력을 수행하고, 화살표 방향에 맞춰 경로를 수정한다.
- **위험 지역 및 계단 지역 수행** : 검은색 바닥의 위험 지역 인식 시 "위험 지역" 음성을 출력하며, 초록색 계단 지역에서는 3층 계단을 정해진 알고리즘엘 따라 등반 및 하강한다.
- **시민 구조** : 위험 지역 내 특정 색상의 물체(시민)를 인식하여 집어 올린 다음 지역 밖으로 안전하게 이동시킨다.
- **종 울리기 및 구조 요청** : 계단 정상에서 종을 울리고 음성으로 구조를 요청하는 미션을 포함한다.
